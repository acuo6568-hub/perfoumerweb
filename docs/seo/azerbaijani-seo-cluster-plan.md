# Azerbaijani SEO Cluster Plan

## Goal

Build a clean Azerbaijani search cluster around fragrance discovery, comparison, notes, and product intent.
The priority is to grow indexable landing pages, strengthen internal link flow, and keep titles tightly aligned with Azeri search behavior.

## Core Cluster Structure

### Pillar pages

| URL | Role | Primary intent |
| --- | --- | --- |
| `/` | Home | Brand discovery, trust, premium fragrance store |
| `/catalog` | Catalog hub | Browse and filter perfumes by brand, note, price |
| `/brands` | Brand hub | Find perfumes by house/brand |
| `/qoxunu` | Scent quiz | Find the right perfume by preference |
| `/compare` | Comparison hub | Compare perfumes side by side |
| `/notes/[slug]` | Note landing pages | Find perfumes by a specific note |
| `/perfumes/[slug]` | Product pages | Exact product intent and conversion |

## Internal Link Map

### Home page links out to

- `/catalog` with anchor: `Kataloqa bax`
- `/brands` with anchor: `Brendlərə bax`
- `/qoxunu` with anchor: `Qoxunu test et`
- `/compare` with anchor: `Ətirləri müqayisə et`
- Top-selling product pages with anchors like `Amber Wood`, `Colonia Intensa`, `Black Incense`
- Key note pages with anchors like `berqamot`, `ağac notları`, `müşk`, `qızılgül`

### Catalog page links out to

- Brand hub pages through brand cards or brand filters
- Product pages through perfume cards
- Note pages through note chips and note filters
- `/compare` for users deciding between two or more scents
- `/qoxunu` for users who need guided discovery

### Brand pages should link to

- Brand-filtered catalog results like `/catalog?brand=Ajmal`
- Related product pages for the brand’s strongest perfumes
- Note pages that match the brand’s common olfactory profile

### Note pages should link to

- Related perfumes carrying the note
- `/catalog?note=[slug]`
- Nearby notes where helpful, such as top/heart/base alternatives
- `/compare` when a user is deciding between similar note profiles

### Product pages should link to

- `/catalog?brand=[brand]`
- Related note pages for top, heart, and base notes
- `/compare` if the product has similar alternatives
- `/wishlist` only for user flow, not for SEO-indexable links

## GSC Submission Checklist

- Verify the property for `https://perfoumer.az/`
- Submit the sitemap at `https://perfoumer.az/sitemap.xml`
- Inspect and request indexing for the home page first
- Then submit the catalog, brands, compare, and scent quiz pages
- Submit product detail pages for the strongest commercial pages first
- Submit note pages next, starting with the most searched notes
- Recheck coverage and indexing after 24 to 72 hours
- Monitor whether Google is choosing the canonical URL you expect
- Fix any duplicate title or duplicate description reports immediately

## Bing Webmaster Tools Checklist

- Verify the `https://perfoumer.az/` site
- Submit `https://perfoumer.az/sitemap.xml`
- Use URL inspection on the same priority order as GSC
- Prioritize canonical product URLs, not filtered query URLs
- Re-submit after any sitemap or title-map changes
- Review crawl errors for product pages and note pages weekly

## Azerbaijani SEO Rules

- Keep titles in natural Azerbaijani first, English only when necessary
- Avoid keyword stuffing and repeated phrase blocks
- Prefer one strong intent per landing page
- Use internal links with descriptive Azeri anchors
- Link to canonical URLs, not filtered duplicates, in indexable content
- Add new cluster pages only when they can support at least 500 to 800 words of useful copy and a clear link path

## Priority Implementation Order

1. Strengthen the home, catalog, brands, compare, and quiz pages.
2. Expand product-page internal links to notes and brand pages.
3. Add new cluster landing pages for men, women, unisex, oud, gift, summer, winter, and longevity intent.
4. Keep submitting canonical URLs in GSC and Bing as new pages go live.
